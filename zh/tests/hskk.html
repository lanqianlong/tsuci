HSKK

HSKK is to measure Chinese learners’ speaking skills. It includes three levels: HSKK (Basic), HSKK (Intermediate) and HSKK (Advanced). Test score is decided by test taker’s on-site performance recorded by the test supervisor.

HSKK can be divided into three levels of HSKK(Basic), HSKK(Intermediate) and HSKK(Advanced), and the test takers' onsite performance will be recorded.

Test Methods

1. HSK includes a paper-based test and an Internet-based test.

  • Paper-based Test: Test takers should fill in all the answers on the Answer Sheet, and the answer should be marked by 2B pencil.
  • Internet-based Test: For HSK iBT, test takers user mouse and keyboard to answer the question. The test time is kept by the iBT system automatically.
2.For HSKK, test takers' answers will be recorded onsite.
  • Test takers should record their answers with his/her respective recording dvices and follow the instructions of the test examiner.
  • Recordings for thest takers in the same test room will start/stop at the same time.

Test Registration and Payment

Test takers can go on the Chinese testing service website (www.chinesetest.cn) to find test dates, the information of the nearest test center or the test center he/she wish to take.

Modification Registration

Before the registration deadline(including the deadline): test takers can change the test date for free. Between the registration deadline and 10 days before the test(including the 10th day): test takers need to pay an amount equal to 30% of the test fee before they can apply for the change during this period. Within 10 days before the test (excluding the 10th day): No application for changes of this kind will be accepted.

Test Regulations

1. Articles Carrier for the test

  • Test admission ticket
  • Identity certificate (copies are invalid) which is authentic, valid, identical with the registration information and carries a photo;
  • 2B pencils (two or more) and eraser, in the event that it is a paper-based test.
2. Test Regulations
  • Test takers should arrive at the test room on time.
  • During the test, test takers must follow the instructions of the test examiner, comply with the test regulations, and complete the test independently.
  • If any test taker conducts cheating acts or diobeying reasonable directions given by the test examiner during the test his/her score will be cancelled.

Test Scores

1. Score enquiry
One month after taking the test test takers can log into the Chinese testing service website and enter his/her test admission ticket number as well as name to view the test scores. 2. Score report
The printed official test score reports will be delivered to the test center 45 days after the test date, and test takers can get it there by presenting their test admission tickets. 3. Validity period of score report.
The test result is a long term available, but the score report to be used ofr the application to Chinese universities and colleges is valid within two years(from the test date). 4. Additional servies concerning score report.
Additional score report and the score report modification service are provided to test takers. Test takers could submit the application on the "Personal information center" page at Chinese testing service website(www.chinesetest.cn).
